# Tutorials

Welcome to the TestLink tutorials! These step-by-step lessons will help you learn how to use TestLink effectively in your PHP projects.

## What You'll Learn

These tutorials are designed for **learning by doing**. Each tutorial guides you through a complete workflow, from installation to advanced patterns.

## Prerequisites

Before starting these tutorials, you should have:

- PHP 8.1 or higher
- Composer installed
- A PHP project with either Pest or PHPUnit (or both)
- Basic understanding of PHP unit testing

## How to Use These Tutorials

1. **Follow in order** - Each tutorial builds on the previous one
2. **Type the code** - Don't just copy-paste; typing helps you learn
3. **Experiment** - Try modifying the examples to see what happens
4. **Ask questions** - If something is unclear, check the [Explanation](/explanation/) section

Ready to begin? Start with [Getting Started](./getting-started)!
